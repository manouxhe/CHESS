##  Objectif
Réaliser une application de gestion des matchs d’une fédération d’échecs.  
Cette application sera utilisée par le personnel administratif de la fédération.  
Ce personnel utilisera l’application pour gérer la base de contacts et d’échanges,  
par mail ou par téléphone, avec les joueurs et les organisateurs de compétitions.
